# Project Brief

## One-Sentence Pitch
一款背包管理类自动战斗游戏。玩家扮演勇者小队冒险路上的后勤大厨，通过食材组合、饭盒填充、饭盒拓展和烹饪构筑，为战士、猎人、法师三位勇者提供各种加成，帮助他们在固定冒险路线中击败六只普通怪物与最终 Boss。

## Core Game Identity
- 类型：背包管理 + 自动战斗
- 玩家身份：勇者小队后勤大厨
- 核心对象：三位角色的独立饭盒、共享食材仓库、六类食物、市场、固定战斗路线
- 核心循环：准备食材（购买或战斗掉落） -> 摆盘 -> 自动战斗 -> 获胜获取特殊食材

## Characters
- 战士
- 猎人
- 法师

## Win / Lose
- 胜利：击败最终 Boss。
- 失败：主角三人血量均扣减到 0，冒险终止，玩家需要从第一关重新开始。
