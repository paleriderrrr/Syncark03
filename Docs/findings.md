## Extracted Summary
- 主玩法是“准备食材 -> 摆盘/整理饭盒 -> 自动战斗 -> 获取食材与金币 -> 进入下一节点”的循环。
- 三名角色分别为战士、猎人、法师，共享食材仓库，但各自有独立饭盒。
- 饭盒初始为 `3x3`，可在市场购买拓展块，允许旋转、留空洞、重新拼接，最终布局需保持连通，且整体受 `6x8` 外框限制。
- 六类食物分别是蔬果、甜品、肉类、饮品、主食、香料，每类有基础属性、羁绊效果、形状和稀有度。
- 羁绊触发规则是同一角色饭盒内出现三个同类食物；该类别总占格数越高，效果层数越高。
- 战斗是自走棋式自动战斗，默认承伤顺序为战士 -> 猎人 -> 法师，单场只对战一个怪物。
- 关卡顺序固定为市场、战斗、休整交替推进，最后进入 Boss 战。
- 市场出售食材和饭盒拓展，食材按稀有度与随机折扣成组出售，刷新价格按斐波那契式递增。
- `食物清单.pdf` 明确了六流派的定位与若干连携关系，更适合作为内容池与设计方向，而非硬数值来源。
- 开发平台在主策划最后一页明确为 `Unity 2022.3.55`。
